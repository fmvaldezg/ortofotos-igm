Cómo usar el mapa

🖱️
Ver información Haz clic sobre cualquier polígono para ver sus metadatos y la URL WMS del servicio.
🗂️
Filtrar por servicio Usa la leyenda para mostrar un solo servicio. Haz clic nuevamente para mostrar todos.
📅
Filtrar por año Selecciona un año en la barra superior para filtrar por fecha de toma. Se puede combinar con el filtro de servicio.
📋
Copiar URL WMS En el popup usa la pestaña iD Editor para obtener una URL template lista para usar en iD. Usa la pestaña JOSM / QGIS para obtener la URL del servicio y el nombre de capa para conectar en JOSM o QGIS.
